# 🛒 WR-Conveniência

Sistema web de e-commerce desenvolvido para simular uma loja de conveniência online, com fluxo completo de compra do usuário.

---

## 🚀 Funcionalidades

✔ Sistema de Login e Cadastro
✔ Catálogo de produtos com layout moderno
✔ Carrinho de compras funcional (com quantidade e total)
✔ Remoção e ajuste de itens no carrinho
✔ Finalização de pedido
✔ Pagamento na entrega (Pix, dinheiro ou cartão)
✔ Tela de pedido com status dinâmico
✔ Notificações automáticas estilo aplicativo

---

## 🧭 Fluxo do Sistema

Login → Cadastro → Produtos → Carrinho → Finalização → Pedido → Produtos

---

## 💻 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (LocalStorage para persistência dos dados)

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com fins acadêmicos para demonstrar conceitos de:

* Interface de usuário (UI)
* Navegação entre páginas
* Manipulação do DOM
* Armazenamento local (LocalStorage)
* Simulação de fluxo real de e-commerce

---

## 📌 Observações

* Não possui integração com banco de dados ainda (simulação local)
* Os dados são armazenados temporariamente no navegador
* Ideal para fins educacionais e demonstração de funcionalidades
